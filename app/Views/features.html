MVC Controllers

Clean separation of concerns with auto-discovery controllers. Handle data fetching, transformation, and business logic elegantly with lifecycle hooks.

BaseController.extend('ProductController', { list: { dataSource: 'storage/data/products.json', transform(products) { return products.map(p => ({ ...p, inStock: p.quantity > 0, priceFormatted: `$${p.price.toFixed(2)}` })); }, afterLoad(products) { console.log(`Loaded ${products.length} products`); } } });

Smart Routing

Hash-based routing with dynamic parameters, active link detection, and smooth anchor scrolling. Supports nested routes and route parameters.

const routes = [ { path: '/', component: 'app/Views/home.html' }, { path: '/products/:id', component: 'app/Views/product-detail.html' }, { path: '/blog/:category/:slug', component: 'app/Views/blog-post.html' } ]; // Navigate programmatically Router.navigate('/products/123'); // Access params in controller dataSource(params) { return `storage/data/products/${params.id}.json`; }

Component System

Reusable HTML components with slots and includes. Build complex UIs from simple, composable pieces with named slots for maximum flexibility.

<!-- Define component: app/Components/card.html --> <div class="card"> <slot name="header"></slot> <slot name="content"></slot> <slot name="footer"></slot> </div> <!-- Use component --> <include src="app/Components/card.html"> <slot name="header"><h2>Title</h2></slot> <slot name="content"><p>Content here</p></slot> <slot name="footer"><button>Action</button></slot> </include>

State Management

Global store with Alpine.js integration. Manage app-wide state that persists across navigation with reactive updates and subscriptions.

// Set global state Store.set('user', { name: 'John', role: 'admin' }); // Get state const user = Store.get('user'); // Subscribe to changes Store.subscribe('user', (newUser) => { console.log('User updated:', newUser); }); // Use in Alpine templates <div x-data> <p x-text="$store.global.user?.name"></p> </div>

Performance

Morphdom for efficient DOM updates, built-in caching, lazy loading controllers, and XSS prevention with DOMPurify. Fast and secure by default.

Morphdom Updates

Efficient DOM diffing preserves Alpine state and minimizes repaints during navigation.

Built-in Caching

Automatic caching of API responses with configurable TTL for optimal performance.

Lazy Loading

Controllers are loaded on-demand, reducing initial bundle size and improving load times.

XSS Prevention

Automatic data sanitization with DOMPurify protects against cross-site scripting attacks.
